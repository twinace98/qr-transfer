```markdown
- {{...}}

{{Link to STATUS.md or paste the relevant section.}}

2. {{...}}

## After-meeting notes

### Discussion
- {{...}}

### Decisions
- {{...}}

### Action items
- [ ] {{owner}} — {{task}} — {{due}}
```
