# References

Literature notes as markdown summaries alongside the corresponding source PDFs.

## Naming

`{first-author-surname-lowercase}-{year}-{topic-slug}.md` paired with `{...}.pdf`.

Examples: `liu-2024-coherent-entropy.md`, `chan-2021-esc-shift.md`.

## Template (per note)

```markdown
# {{First-author}} {{year}} — {{topic}}

**Citation**: {{full citation}}
**DOI / arXiv**: {{...}}
**Read on**: {{YYYY-MM-DD}}

## Why this paper matters here

{{One paragraph — what decision in *our* project does this inform?}}

## Method summary

{{Parameters, algorithm, code used.}}

## Key results

{{Numbers / figures we would want to reuse or compare against.}}

## Transferable to our project

- {{specific parameter / method choice worth adopting}}
- {{specific result to benchmark against}}

## Caveats / things that don't transfer

- {{...}}
```
