# Scripts

Project-wide utilities (not specific to one `data/` case).

Examples from past projects:
- `check_latex_spacing.py` — enforces markdown LaTeX inline-math rule (`scripts/check_latex_spacing.py`).
- `download_pseudos.sh` — provisions external dependencies used across cases.
- `build_structures.py` — generates inputs consumed by multiple `data/` directories.

Case-specific scripts (one-shot input generators, per-case plotters) live in their `data/{nnn}-.../` directory, not here.
