# Reports

One subdirectory per topic. Each topic's main document is `{topic}/{topic}.md`, with figures and plot scripts alongside.

Reports are **periodic accumulators**, not final deliverables. They grow at each decision gate and at phase completion.

## Report template

Each `{topic}/{topic}.md` should have these sections:

- **Key Finding** — one paragraph, the headline.
- **Method** — what was run, parameters, data paths.
- **Results** — tables, figures (relative paths: `![caption](figure.png)`).
- **Discussion** — interpretation, comparison to expectation/experiment/literature, caveats.
